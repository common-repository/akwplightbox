It makes all the image in a post that are linked to larger images, to open with lightbox effect.
== Description ==
Aim of this plugin was to create a simple lightbox, to be used with the images on the post, it helps you add your images as you do now, it will take care of rest of the things. It works transparently without adding any extra work on your part.

For more information check out the <a href="http://amiworks.co.in/talk/akwplightbox/" title="akWPLightbox : simple lightbox for wordpress" >plugin homepage</a>.

Tip: To keep track of this plugin's evolution and features, you are advised to <a href="" title="Grab Amiworks RSS Feed">subscribe to the author's feed</a>
== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `akwpLightbox` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it nothing else.
1. Now it will automatically find the images in the post and show it with lightbox effect.
== Frequently Asked Questions ==

= How to use it? =

Simple, just activate the plugin and you are done. just remeber to link the images that you post in blog to the image that you want to see in lightbox.
